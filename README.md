Este proyecto procesa 4 datasets del experimento del efecto Zeeman.
Incluye preprocesamiento, correlaciones y regresión lineal (uni y multivariable).
> Nota: los resultados no son concluyentes respecto al desdoblamiento de líneas; se sugiere en trabajos futuros medir posiciones/anchos de línea para modelar vs campo magnético.

## Estructura
- `notebooks/`: análisis principal (`EfectoZeeman.ipynb`).
- `data/`: datos del experimento.
